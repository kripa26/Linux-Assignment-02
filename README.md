# Linux Fundamentals Assignment

This repository holds my submissions for Linux Task 02.

## Contents of this Repository
- **`linux_task_02_solution.md`**: My step-by-step documentation and commands used to complete the assignment, covering user creation, group assignments, and file ownership.
- **`linux_task_02_diagram.md`**: A visual mind map that illustrates the relationships between the users, their groups, and the permissions applied to the files.

## Summary of Topics
* Creating and managing groups and users (`useradd`, `groupadd`, `usermod`)
* Understanding system identifiers (`UID`, `GID`, `/etc/passwd`)
* Managing file permissions and ownership (`chmod`, `chown`, octal values)
